# Task 2 - Titanic Dataset Analysis

## Objective

Analyze the Titanic dataset to identify factors affecting passenger survival.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset

Titanic Dataset containing passenger information such as age, gender, passenger class, fare, and survival status.

## Data Cleaning

* Filled missing values in Age using median.
* Filled missing values in Embarked using mode.
* Removed Cabin column due to a large number of missing values.

## Analysis Performed

1. Survival Rate by Gender
2. Survival Rate by Passenger Class
3. Survival Rate by Age Group
4. Passenger Age Distribution

## Key Findings

* Female passengers had a higher survival rate than male passengers.
* First-class passengers had the highest survival rate.
* Passenger class significantly influenced survival chances.
* Most passengers were between 20 and 40 years old.
* Age played a role in survival patterns.
